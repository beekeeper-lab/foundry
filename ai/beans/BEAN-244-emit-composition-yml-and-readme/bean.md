## Problem Statement

The generated `validate-repo` skill (`ai-team-library/claude/skills/validate-repo/SKILL.md`) checks for two artifacts at the generated project root that the scaffold never produces:

- `ai/team/composition.yml` — the source-spec snapshot that enables overlay/regeneration workflows, audit ("what team was this built with?"), and deterministic round-trip editing. Verifiable on `test` branch: regenerating `examples/small-python-team.yml` produces no `ai/team/` directory at all.
- `README.md` — a starter readme at the project root.

A freshly generated project fails its own validator on these two paths. That's a framework-integrity failure: the validator is authoritative documentation of the expected project shape.

Both artifacts are genuinely valuable (not vestigial). `composition.yml` is not redundant with `manifest.json` — manifest is generated JSON metadata; `composition.yml` is the source spec, round-trippable. `README.md` is a standard project-root convention that should not require the user to create by hand.

## Goal

The scaffold emits both `ai/team/composition.yml` and `README.md`. A freshly generated project passes `validate-repo`'s structural checks (for these two artifacts) with zero errors.

## Scope

### In Scope
- Scaffold / generator emits `ai/team/composition.yml` — a YAML serialization of the `CompositionSpec` that produced the project (distinct from JSON manifest in structure and purpose).
- Scaffold / generator emits `README.md` at the project root with a minimal starter template (project name, one-line description, quick-start pointer to `CLAUDE.md`, "generated by Foundry" footer).
- Unit tests covering both emissions (content, location, YAML shape for composition.yml).
- BEAN-242's structural assertion passes after this ships.

### Out of Scope
- Changing the manifest.json format or location.
- Modifying `ai-team-library/claude/skills/validate-repo/SKILL.md` — the skill's expectations are already correct; the scaffold needs to catch up.
- Placeholder leakage (BEAN-243) or CLAUDE.md shape (done by BEAN-233).
- Implementing a full runnable validate-repo skill in Python.
