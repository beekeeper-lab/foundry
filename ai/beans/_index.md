| Status | Meaning |

| Category | Meaning |

| Process | Changes to the AI team workflow — agent instructions, skills, commands, communication patterns (`.claude/`, `ai/`) |
| Infra | Git workflow, hooks, branch protection, CI/CD, deployment |
