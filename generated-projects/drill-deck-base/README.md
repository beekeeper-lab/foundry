## Generating images and audio

This project ships with two plan-driven media skills. The plan files at the project root are the review surface; on-disk MP3s and PNGs are the output.

- `IMAGE-PLAN.md` — image entries with provider/quality frontmatter; the
  `generate-image` skill loops the plan, writes one PNG per entry, and
  skips entries already present on disk.
- `NARRATION-PLAN.md` + inline `> 🎙️ ...` blockquotes in your source
  markdown — `generate-audio` walks the source files, generates one MP3
  per block, and writes a per-source `manifest.json` storing the
  stripped text for content-hash dedup across pages.

Both skills discover `.env` from cwd → parents → `$HOME`. Required env vars (set the ones for the providers you use):

- `GEMINI_API_KEY` (default image provider)
- `OPENAI_API_KEY` (when `**Generator:**` selects an OpenAI model)
- `ELEVENLABS_API_KEY` (all audio runs)

See `.claude/shared/skills/generate-image/SKILL.md` and `.claude/shared/skills/generate-audio/SKILL.md` for usage and cost discipline notes.
